Property vim4-annotations:generalQuantity

dc:description

Annotation subject is a scalar property; annotation object is an ISO/IEC vim4:GeneralUnitaryQuantity instance.

rdf:type

owl:AnnotationProperty

rdfs:comment

The annotated scalar property corresponds to a vim4:IndividualUnitaryQuantity that vim4:characterizes a vim4:Object corresponding to the domain of the annotated scalar property.
